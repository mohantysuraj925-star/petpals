# 🌿 Green Earth

Innovation Week Project built with React and Tailwind CSS.

## About Project

Green Earth is an environmental awareness website that motivates people to:

🌱 Plant Trees  
💧 Save Water  
♻️ Reduce Pollution  
🌍 Protect Earth  

## Features

- Responsive design
- Smooth scrolling navigation
- Animated sections
- Green environmental theme
- Contact and Green Pledge section

## Technology Used

- React + Vite
- Tailwind CSS
- Framer Motion
- React Icons
- Lucide Icons
- React Scroll

## Run Project

Install:

npm install

Start:

npm run dev
